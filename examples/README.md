# Examples

This directory contains examples demonstrating how to use sdcvalidatorRust.

## Status

Examples will be added during implementation (Q2 2026).

## Planned Examples

- `basic_validation.rs` - Simple validation example
- `recovery_mode.rs` - Using recovery mode with ExceptionalValues
- `cli_usage.sh` - Command-line interface examples
- `custom_error_handling.rs` - Custom error classification
- `streaming_validation.rs` - Validating large documents

## Running Examples

Once implemented, run examples with:

```bash
cargo run --example basic_validation
```

## Documentation

Each example will include comprehensive documentation explaining the concepts and usage patterns.
